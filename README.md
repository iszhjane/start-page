# zzhstart-page

Forked 自 [ahmetdem/start-page](https://github.com/ahmetdem/start-page) —— 感谢原作者的精美起点。

> 一个功能完善的浏览器起始页，支持书签管理、自定义主题和跨设备同步。

---

## 🆕 基于上游的新增功能

### 🎨 主题系统
- **默认主题** — 简洁现代，搭配 squircle 圆角设计
- **像素主题** — 8-bit 黑白色调，绿色高亮，像素字体与硬阴影
- **浅色 / 深色模式** 切换（像素主题为单色模式）

### 🔖 书签管理
- **完整 CRUD** — 在设置面板中增、删、改书签
- **批量导入** — 批量粘贴网址，支持带标题和分类
- **分组管理** — 创建、重命名、排序、删除分组
- **书签排序** — 按名称或添加时间排序
- **批量操作** — 勾选多个书签，批量移动分组或删除
- **长按操作** — 主页长按书签唤出编辑/删除按钮
- **分页翻页** — 每页 20 个书签，横向滑动 + 圆点指示器

### 🔍 搜索与命令
- **终端风格输入框** 带打字动画
- **多搜索引擎**：Google、Bing、DuckDuckGo、Brave、自定义
- **工作流** — 保存一组 URL，一个命令全部打开
- **特殊命令**：`r:<query>`、`m:<query>`、`add <url> [title] [category]`

### ⚙️ 设置面板
完整的设置界面，共 7 个标签页：

| 标签页 | 功能 |
|--------|------|
| 主题 | 主题卡片（默认 / 像素），浅色/深色模式 |
| 书签 | 增删改、批量导入、排序、勾选移动/删除 |
| 分组 | 创建、重命名、排序、删除分组 |
| 工作流 | 创建命名的 URL 集合，一键全部打开 |
| 搜索引擎 | 选择默认引擎，配置自定义引擎 |
| 快捷键 | 查看所有键盘快捷键 |
| 同步 | 邮箱登录/注册/绑定、手动同步、状态指示 |

### 🔄 跨设备同步（Supabase）
- **匿名登录** — 首次访问自动静默登录（无需注册）
- **邮箱注册/登录** — 绑定账号以跨设备同步数据
- **自动同步** — 每次数据变更 800ms 防抖推送
- **手动同步** — 设置面板中的"手动同步"按钮（双向：先拉后推）
- **状态指示器** — 页面右上角显示 ✓ 绿色 / ✗ 红色 / ⚡ 离线 / ⟳ 旋转

## 🚀 部署

Fork 本仓库并部署到任意静态托管平台（Vercel、Netlify、GitHub Pages 等）。

### 可选：启用数据同步

1. 创建一个 Supabase 项目并执行 SQL schema（参见 `supabase-schema.sql`）
2. 复制 `.env.example` 为 `.env`，填入 Supabase 凭据
3. 重新部署
